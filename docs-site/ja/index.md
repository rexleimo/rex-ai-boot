# rex-ai-boot ドキュメント

`rex-ai-boot` は次の CLI 向けローカルワークフローレイヤーです。

- Codex CLI
- Claude Code
- Gemini CLI

追加する機能は 2 つです。

1. ファイルシステム ContextDB（セッション記憶）
2. 透過ラッパー（`codex` / `claude` / `gemini` をそのまま利用）

## すぐに試す

```bash
codex
claude
gemini
```
