# rex-ai-boot 문서

`rex-ai-boot`는 다음 CLI를 위한 로컬 워크플로 레이어입니다.

- Codex CLI
- Claude Code
- Gemini CLI

핵심 추가 기능:

1. 파일시스템 ContextDB (세션 메모리)
2. 투명 래퍼 (`codex` / `claude` / `gemini` 그대로 사용)

## 빠른 실행

```bash
codex
claude
gemini
```
